# ⚽ Análise de Público do Brasileirão 2017

## Objetivo

Investigar os dados de público do Campeonato Brasileiro de 2017 utilizando Python e Pandas.

---

## Perguntas respondidas

- Qual foi o maior público registrado em 2017?
- Qual foi o menor público registrado em 2017?
- Quais foram os 10 jogos com maior público?

---

## Tecnologias utilizadas

- Python
- Pandas
- Google Colab
- Git
- GitHub

---

## Estrutura do projeto

analise-publico-brasileirao-2017/
├── dados/
├── notebooks/
└── README.md
---

## Principais resultados

- Identificação do maior público do campeonato;
- Identificação do menor público;
- Ranking dos 10 jogos com maior presença de torcedores.
